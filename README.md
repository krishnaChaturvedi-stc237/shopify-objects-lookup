# Shopify Liquid Object Lookup

The ultimate IntelliSense companion for Shopify 2.0 Theme Development. Get instant suggestions for Liquid objects and their properties with direct links to official documentation.

## Features

- **Object Autocomplete**: Suggests global Shopify objects (product, cart, collection, etc.).
- **Property Discovery**: Type a `.` after an object to see all available properties.
- **Rich Documentation**: Hover over suggestions to see descriptions and usage examples.
- **Quick Reference**: Direct links to Shopify's official documentation for every property.

## Usage

1. Open a `.liquid` or `.html` file.
2. Type a Shopify object like `product`.
3. Type a dot `.` to see properties: `product.title`.

## Supported Objects
- Product
- Cart
- Variant
- Line Item
- Collection
